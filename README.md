# Nome do Projeto
Diário da formação
Página web contruida em Reveal.js com código de suporte que reporta ao trabalho desenvolvido no âmbito de uma formação de desenvolvimento profissional e pessoal.

## Estrutura do repositório

- index.html — apresentação Reveal.js (ficheiro principal)
- css/ — estilos
- js/ — scripts e código
- plugin/ — plugins usados
- media/ — imagens e outros assets
- LICENSE-CODE — licença aplicável ao código (MIT)
- LICENSE-ASSETS — licença aplicável aos assets (CC BY-NC-SA 4.0)
- THIRD_PARTY_NOTES.md — inventário de terceiros e respetivas licenças
- README.md — este ficheiro
